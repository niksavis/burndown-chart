# HTML Reporting Guide

**Audience**: Users generating stakeholder reports and developers maintaining report templates
**Part of**: [Documentation Index](readme.md)

## Overview

HTML reporting generates a self-contained report that can be opened offline in any browser. Reports are designed for stakeholder sharing and archive snapshots.

## Report Contents

- Summary metrics for dashboard, DORA, and Flow.
- Charts rendered as embedded Plotly content.
- Export timestamp and profile context.

## Output Characteristics

- Single HTML file with embedded assets.
- No external dependencies required to view.
- Suitable for email or file sharing.
